### My Git Workflow
This project strictly follows a `dev` -> `main` branching model:

* **`main` branch:** This is the production-ready branch. It only contains stable, finished code that has been reviewed and merged.
* **`dev` branch:** This is the primary development branch. All new work, features, and fixes are created here. Code is only merged into `main` from this branch via a Pull Request.
